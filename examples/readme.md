| 例程类型    | 说明                                                         |
| ----------- | ------------------------------------------------------------ |
| application | 应用例程，具有具体应用场景的应用方案，如 AWTK、 web 服务器、 shell 应用等 |
| component   | 组件的基本使用例程，如文件系统、网络等                       |
| feature     | CPU 特殊外设例程，如浮点运算、中断响应测试等例程             |
| peripheral  | 基础外设例程，如 SPI、 I2C、 CAN、 ADC 等                    |
| system      | 系统内核例程，如任务、消息、信号量、软件定时器等             |
